# GraphQL Profile — Tomorrow School

A single-page profile app built with vanilla JS and GraphQL.

**Live:** https://nurikshuu.github.io/graphql/

## Features

- Login with school credentials (JWT auth)
- Overview: profile info, XP summary, audit ratio, recent activity
- Statistics: 6 SVG charts — XP timeline, top projects, results donut, 2× skills radar, monthly XP
- Error pages: 400, 401, 403, 404, 500

## Stack

Vanilla JS · SVG charts · GraphQL (no libraries)

## Run

Open `index.html` in a browser or visit the live link above.
